# CryptRGR
Multi-file Encryption/Decryption Programm

Реализация комплексной системы шифрования данных на языке C++. Проект объединяет в себе четыре различных алгоритма шифрования: от классических симметричных методов до современных асимметричных криптосистем.

## Возможности
* Шифрование и дешифрование текста
* Шифрование и дешифрование файлов
* Генерация ключа
* Выбор алгоритма через консольное меню
* Динамическая загрузка алгоритмов из папки `algorithms/`
* Подключение каждого алгоритма как отдельной `.so` *(Linux, MacOS)* или `.dll` *(Windows)* библиотеки

# Архитектура проекта 
Структура проекта мультифайловая
Проект представлен в виде следующих файлов:

## main & динамический загрузчик
* `main.cpp` — Точка входа, организация пользовательского интерфейса, управление потоком данных
* `algorithm_interface.h` — Хэдер интерфейса криптомодулей с объявлением экспортируемых функций
* `library_loader.h` / `library_loader.cpp` — Обеспечивают загрузку библиотек модулей

## Подключаемые Криптомодули
Каждый шифр компилируется в виде независимого .so файла
* `xor.cpp` - Побитовое шифрование
* `gost.cpp` — Симметричное блочное шифрование по ГОСТ 28147-89
* `elgamalfull.cpp` — Асимметричная система шифрования на основе дискретного логарифма
* `shamir.cpp` — Схема разделения секрета/бесключевого шифрования Ади Шамира

# Поддерживаемые шифры и детали их реализации
### 1. XOR Шифрование `[xor.cpp]`
Симметричный метод побитового шифрования с использованием ключа, основанный на логической операции исключающего «ИЛИ». 

### 2. ГОСТ 28147-89 (Режим простой замены) `[gost.cpp]`
Симметричный блочный шифр (блок 64 бита, 32 раунда). Использует таблицы замен S-Box и Сеть Фейстеля. Для выравнивания некратных данных применяется паддинг **PKCS#7**, текстовый вывод кодируется в **HEX**.

### 3. Схема Эль-Гамаля (Упрощённая) `[elgamalfull.cpp]`
Асимметричная криптосистема на основе сложности дискретного логарифмирования. Вероятностный шифр (использует случайное сессионное число k для каждого байта). Реализовано быстрое возведение в степень по модулю и расширенный алгоритм Евклида.

### 4. Схема разделения секрета Шамира `[shamir.cpp]`
Криптографическая схема, где каждый байт — свободный коэффициент полинома над конечным полем по модулю простого числа P. Секрет разделяется на 5 долей, для восстановления (дешифрования) достаточно любых 3 долей с использованием интерполяции Лагранжа.
